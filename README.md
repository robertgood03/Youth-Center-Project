# Youth Center Project

A web application for managing youth center activities, programs, and memberships.

## Features

- Member registration and management
- Program scheduling
- Event calendar
- Attendance tracking
- Parent/Guardian portal

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/robertgood03/Youth-Center-Project.git
   cd Youth-Center-Project
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
